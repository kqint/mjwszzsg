# 编译打包指南

## 前置条件

- Java JDK 8+
- apktool 3.0.1（`apktool.jar` 放在项目根目录）
- 签名用的 `debug.keystore`（项目根目录已有）

## 完整打包流程

### 1. 编译 APK

```bash
cd d:\Dev\AndroidDev\mjwszzsg
java -jar apktool.jar b -o mjwszzsg_fixed.apk .
```

输出：`mjwszzsg_fixed.apk`

### 2. 签名

```bash
jarsigner -verbose -keystore debug.keystore -storepass android \
  mjwszzsg_fixed.apk androiddebugkey
```

### 3. 安装到设备

```bash
adb install -r mjwszzsg_fixed.apk
```

## 一键命令

cd d:\Dev\AndroidDev\mjwszzsg && java -jar apktool.jar b -o mjwszzsg_fixed.apk . && jarsigner -verbose -keystore debug.keystore -storepass android mjwszzsg_fixed.apk androiddebugkey && adb install -r mjwszzsg_fixed.apk

## 验证

```bash
# 验证签名
jarsigner -verify -verbose mjwszzsg_fixed.apk

# 查看 APK 中的 Manifest
java -jar apktool.jar d -s -f mjwszzsg_fixed.apk -o /tmp/check && cat /tmp/check/AndroidManifest.xml && rm -rf /tmp/check

# 查看 APK 中的 targetSdkVersion
java -jar apktool.jar d -s -f mjwszzsg_fixed.apk -o /tmp/check && grep targetSdkVersion /tmp/check/apktool.yml && rm -rf /tmp/check
```

## 调试

```bash
# 清空日志并启动崩溃监控
adb logcat -c
adb logcat -b crash | grep -E "AndroidRuntime|FATAL|F/libc|tombstone"

# 查看完整日志
adb logcat > crash_log.txt

# 抓取 native 崩溃堆栈
adb logcat | grep -E "DEBUG|backtrace|signal"
```

## keystore 信息

| 参数 | 值 |
|:--|:--|
| 文件 | `debug.keystore` |
| 别名 | `androiddebugkey` |
| 密码 | `android` |
| 算法 | RSA |
| 有效期 | 10000 天（~27 年） |

### 生成 keystore（一次性）

```bash
keytool -genkey -v -keystore debug.keystore -alias androiddebugkey \
  -storepass android -keypass android -keyalg RSA -validity 10000 \
  -dname "CN=Android Debug"
```

## 常见问题

### smali 编译失败

```
I: Smaling smali folder into classes.dex...
Exception in thread "main" org.jf.smali.main: ...
```

通常是 smali 语法错误（寄存器冲突、标签重复等）。检查最近修改的 smali 文件。

### .so 压缩导致闪退

`apktool.yml` 的 `doNotCompress` 中必须包含 `- so`，否则 .so 文件会被压缩导致 `UnsatisfiedLinkError`。

### 签名后仍然提示未签名

使用 `jarsigner -verify` 确认签名状态。部分国产 ROM 需要额外的签名验证。

### adb install 失败 `INSTALL_FAILED_UPDATE_INCOMPATIBLE`

签名变化或包名冲突。先卸载旧版：
```bash
adb uninstall com.gamedo.v360
adb install mjwszzsg_fixed.apk
```
